2/5
Foundation

Growing Agents is the core

The foundational insight: agents that grow over time have compounding value. The growth comes from three feedback loops:

Context Loop — Bringing context back into the agent. Each interaction adds to growth.
Integration Loop — Local file access (ClodBot/Letabot). Proprietary data at scale.
Self-Learning Loop — Agents learning from interactions, improving across the entire cohort.

Token costs will drop. Infinite agents become viable. Continuity workflows and multi-agent communication at scale.

3/5
Key Discovery

The Specialized Agent Problem

Specialized agents have magic. But the magic comes from chat history growth — not memory blocks.

You can't replicate a specialized agent by copying its memory. The unknown property is in how context accumulates through conversation.

Scaling Question

How do you scale something whose value is in undivided history?

Two directions being explored:

1. Supervisor → Sub-agent hierarchy — learnings flow up, updates push down, push model back to supervisor's chat history

2. Horizontal sharding — partition context, filter by user, skip consistency overhead (requires high-level filtering)
