Task Spec 1

Commitment Status Check

DEADLINE_APPROACHING
TRIGGER: 48hrs before deadline OUTPUT: Prompt to founders PRIORITY: Medium
Ask founders for status update on approaching commitment. Do NOT alert mentor yet.
PROMPT TEMPLATE
"Desktop app deadline is Feb 25-28. Current status: ⏳ Not started Update? Reply with: Shipped / In progress / Blocked (reason)"
Edge Case: No Response in 24hrs
Escalate to mentor: "Jai & Satith haven't updated status on [commitment]. May need check-in."

Task Spec 2

Deadline Missed Flag

DEADLINE_MISSED
TRIGGER: Deadline passed + no status update OUTPUT: Alert both parties PRIORITY: High
Surface the mismatch. Don't scold — just make it visible.
TO FOUNDERS
"Desktop app was due Feb 25-28. No status update received. What happened? (This helps Craig coach you better.)"
TO MENTOR
"Jai & Satith missed deadline: Ship desktop app. Possible patterns: perfectionism, scope creep, blocker. Ask in next session: 'What got in the way?'"

Task Spec 3

Weekly Hypothesis Prompt

HYPOTHESIS_CHECK
TRIGGER: Every Friday 9am OUTPUT: Prompt to founders PRIORITY: Low
Force learning mode. Reject task-list answers.
PROMPT TEMPLATE
"This week, what evidence did you get about: [HYPOTHESIS] Answer format: ✓ Confirmed — what did users say/do? ✗ Disconfirmed — what surprised you? ? Still unclear — what would test it?"
Edge Case: Task List Answer
If founders reply with tasks ("we added X, we built Y"), respond:
"Those are tasks. What did users say about the NPC interactions?"

Task Spec 5

Implicit Commitment Surface

IMPLICIT_NUDGE
TRIGGER: Bi-weekly, low priority OUTPUT: Soft prompt to founders PRIORITY: Low
Keep implicit commitments visible without nagging. Gentle reminders.
PROMPT TEMPLATE
"Quick check on the back-burner items: • Context-driven gameplay — any exploration? • API/library pivot — still thinking about it? • Marc Andreessen email — ready to send? Reply only if there's an update. Otherwise, ignore."

Rule: These are soft. No escalation. No mentor alert. Just visibility.

PATTERN_FLAG

Detect patterns that humans miss. Alert mentor, not founders.

ALERT TEMPLATE
"Pattern detected: [PATTERN_NAME] Occurrences: N Sessions: [dates] Suggested intervention: [coaching lever]"
