The Real Opportunity

Test the Blocker, Don't Build Around It

I keep falling into the same pattern:

  • October: Wanted to build a website before getting customers
  • November: Wanted to build anonymous upload before asking for payment
  • December: Saying I need to "work on the anonymous part"

But I can hack anonymity with existing tools:

  • Google Form with no email collection + code word
  • Venmo/CashApp payment with just their code
  • Results delivered via code word lookup

No building required. Just testing.
