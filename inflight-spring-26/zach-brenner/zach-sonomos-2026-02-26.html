Where I Started

Last week, Craig and Neal pushed me: "Go install it yourself. Don't leave until they're running."

I did two Collison installs. And learned something fast.

Install #1

Had to use developer mode. Lawyer said "fix the install process before you come back."

The app works. The install doesn't.

Install #2

She likes it. But asked for Cloak. Noticed false positives. Suggested $4.99 price.

Same pattern in every conversation: everyone wants Cloak, everyone thinks $7.99 is too high for Dagger alone.

2/5

What I Discovered

Three Signals, Same Message

The Friction

  • Developer mode install kills momentum
  • Lawyers want simple, not technical
  • Chrome Web Store is the only path

The Feedback

  • "I'd buy once Cloak is ready"
  • "$4.99 feels right for current features"
  • "Small and medium firms need this"

The Architecture Insight

Dagger and Cloak share the same engine. Dagger wasn't a delay — it was the foundation. Now the engine exists, Cloak ships faster.

Users don't see the foundation. They just see the missing feature.

3/5

The Real Opportunity

The Decision

Not $7.99. Not $4.99. $3.99.

Why? Because I'm not trying to maximize revenue right now. I'm maximizing users who experience the core value before Cloak ships.

Positioning

"Core protection now. Cloak coming soon."

Not "early access." Not "beta." Just: here's what you get, here's what's coming.

Urgency Angle

"Lock in $3.99 before Cloak ships."

The pricing becomes a feature: early adopters get grandfathered. Cloak justifies the raise to $7.99.
